# Название проекта: AI Task 3

## Обзор
Этот проект — приложение на FastAPI, которое предоставляет эндпоинты для проверки работоспособности, предсказаний с использованием обученной модели и информации о модели. Используется сериализованная модель машинного обучения, хранящаяся в каталоге `models`.

## Структура проекта
```
task_3
├── app
│   ├── server.py          # Приложение FastAPI с эндпоинтами
│   └── ml_tools.py        # Утилиты для работы с моделью
├── models
│   └── model.joblib       # Сериализованная модель
├── Dockerfile             # Описание Docker-образа
├── requirements.txt       # Зависимости Python
├── .dockerignore          # Файлы, игнорируемые при сборке Docker
└── README.md              # Документация проекта
```

## Установка

1. **Склонируйте репозиторий**:
   ```
   git clone <repository-url>
   cd task_3
   ```

2. **Создайте виртуальное окружение** (рекомендуется):
   ```
   python -m venv venv
   source venv/bin/activate  # На Windows используйте `venv\Scripts\activate`
   ```

3. **Установите зависимости**:
   ```
   pip install -r requirements.txt
   ```

4. **Запустите приложение FastAPI**:
   ```
   uvicorn app.server:app --reload
   ```

   Приложение будет доступно по адресу `http://127.0.0.1:8000`.

## Использование

- **Проверка работоспособности**:
  - Эндпоинт: `/ping`
  - Метод: GET
  - Ответ: `{"message": "ok"}`

- **Предсказание**:
  - Эндпоинт: `/prediction`
  - Метод: POST
  - Тело запроса: JSON с признаками
  - Пример:
    ```json
    {
      "feature_1": 1.0,
      "feature_2": 2.0,
      "feature_3": 3.0,
      "feature_4": 4.0,
      "feature_5": 5.0
    }
    ```
  - Ответ: Результат предсказания модели.

- **Информация о модели**:
  - Эндпоинт: `/model_info`
  - Метод: GET
  - Ответ: Метрики оценки модели.

## Docker

Сборка и запуск контейнера Docker:

1. **Собрать образ**:
   ```
   docker build -t ai-task-3 .
   ```

2. **Запустить контейнер**:
   ```
   docker run -d -p 8000:8000 ai-task-3
   ```

Приложение будет доступно по адресу `http://localhost:8000`.
